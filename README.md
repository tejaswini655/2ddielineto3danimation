# 2D Dieline to 3D Box Animation

This project converts a 2D packaging dieline into an interactive 3D folding box using React.js and Three.js.

## Features
- Upload a dieline image
- View the 2D layout
- Animate folding and unfolding
- Interactive 3D controls
- Smooth box animation

## Technologies Used
- React.js
- Three.js
- React Three Fiber
- React Three Drei
- Vite

## How to Run

```bash
npm install
npm run dev
